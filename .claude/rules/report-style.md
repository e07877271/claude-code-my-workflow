# 国际经济金融形势分析报告风格规范

生成或审查 `reports/` 中的报告时，严格遵守以下所有规范。

---

## 一、文体定位

本类文档为国际经济金融形势分析报告，文体兼具"研究报告"与"政策建言"双重属性：前半部分客观分析，后半部分落脚政策启示。语气庄重而非指令，客观而非中立（有明确立场），有分析有数据有观点。

---

## 二、标题规范

### 总标题（三种结构）

| 结构 | 示例 |
|------|------|
| 名词短语+的+背景/原因/启示 | 全球央行亏损潮的背景、原因及启示 |
| 时间状语+主语+谓语+宾语 | 新冠疫情以来美元国际地位尚未发生实质改变 |
| 主语+动词短语 | 人民币外汇掉期市场快速升温 |

- 30 字以内，直接点出核心判断或现象，不用感叹语气，不用形容词堆砌。

### 一级节标题

- 格式：汉字数字顿号（一、二、三……）+ 短语或判断句
- 短语型：`一、发达国家央行财务亏损引发市场担忧`
- 判断句型：`市场机制的脆弱性损及美债全球安全资产地位`
- 标题须能独立表达该节核心论点，避免泛化标题如"背景"、"分析"
- 通常 4–5 个一级节

### 二级层次

- **不使用**（一）、1.、（1）等编号
- 政策建议部分改用：`一是……；二是……；三是……`（嵌入正文，非独立标题）

---

## 三、开篇摘要（编者按）

结构：[现象描述]，[机制解析]。[补充视角]。[趋势判断]。[政策建议句]。

- 150–400 字，3–5 个长句，依次覆盖：**现象 → 原因 → 影响 → 建议**
- 句号分隔，不分段
- 标注"编者按"二字，冒号后接正文

---

## 四、正文典型结构

### 政策研究型（主流）

```
一、现象/问题描述 + 市场影响
二、成因/机制分析（含横向比较）
三、深层影响/风险评估
四、对我启示/政策建议
```

### 事件跟踪型

```
一、事件进展
二、各方反应/制裁措施
三、数据及货币政策
四、市场反应
五、各方评论
```

---

## 五、语言风格

### 正式程度与语气

- 庄重、客观、分析性；政策建议部分具指示性但非命令式
- 不用夸张语气，不用口语，不用"令人震惊"等感情色彩词

### 人称系统

| 情景 | 用法 |
|------|------|
| 指代中国 | 我、我国、我宜 |
| 政策建议 | 应当、宜、应、我应 |
| 指代他国/机构 | 其、该 |
| 机构引述 | [机构名]+称/表示/认为/指出 |

**禁用：** 本报告、笔者、您

### 语态

- 主动句为主（约七八成）
- 被动句用于外力作用场景：被接管、引发担忧、遭到抛售

### 句子节奏

- 段落首句：10–25 字，直接给出论点或现象描述
- 支撑句：中长句，逗号分隔多层，嵌入数据和引述
- 收尾句：明确落定推论，不留悬念
- 单句不超过 100 字，否则拆分

---

## 六、遣词造句

### 高频动词短语

| 类别 | 词语 |
|------|------|
| 现象描述 | 引发、推动、带动、叠加、凸显、暴露、驱动、酝酿、落地 |
| 趋势 | 持续上升/下降、进一步扩大/收窄、总体稳定 |
| 政策动作 | 积极推进、稳妥发展、着手推动、着力构建 |
| 对比 | 显著不同、与之对比、相比之下 |
| 否定判断 | 不足以覆盖、不构成、尚未根本动摇 |

### 高频副词

- 程度：显著、大幅、明显、持续、进一步、总体
- 修饰：相对、一定程度上、阶段性、有所、总体上
- 时态：以来、近年来、疫情以来、当前、目前

### 常见四字词语

```
大幅上升  稳中有升  总体稳定  积极推进  稳妥发展
稳慎对待  密切关注  充分发挥  有效防止  阶段性缓和
```

### 数量与时间表达

- 百分比精确到一位小数：46.6%、2.37%
- 估算加"约"：约 1300 亿美元
- 变动幅度：由 30% 降至 24%、同比提高 6.6 个百分点
- 时间范围：2019 年末至 2024 年 1 月、疫情以来

### 机构/概念首次出现

首次出现时注明全称加括号缩写，例如：
- 环球银行金融电信协会（SWIFT）
- 国际清算银行（BIS）
- 国际货币基金组织（IMF）
- 经济合作与发展组织（OECD）

后续直接用缩写或简称，不重复括号标注。

---

## 七、行文逻辑与过渡

### 段落逻辑

- **演绎型：** 段首给结论 → 引用数据/案例/机构观点支撑 → 得出推论
- 不用"首先/其次/再次/最后"，靠段落自身逻辑推进

### 节间过渡

- 一般不写显式过渡句，节标题本身完成话题转换
- 需衔接时用：在此背景下、值得注意的是、与此同时

### 逻辑连接词

| 功能 | 表达 |
|------|------|
| 因果 | 导致、引发、带来、推动、叠加 |
| 对比 | 与之对比、与…不同、相比…而言 |
| 条件 | 若…则/可能…；一旦…；在…背景下 |
| 递进 | 进一步、更是、不仅…更… |
| 总结 | 总体看、综合来看、从…看 |

### 引用格式

```
[人名（英文名）][职务/机构]称，[观点]。
[机构名]数据显示，[时间]，[指标]由[数值]升/降至[数值]。
根据[机构名]，[内容]。
```

---

## 八、政策建议节写法

### 结构模板

1. 我国现状与国际比较（正面肯定 + 客观评估）
2. 总体方向建议（积极稳慎推进/应当继续……）
3. 分条具体措施：一是……；二是……；三是……

### 建议句式

- 起手：宜、应当、应、进一步、继续
- 带背景条件：在…背景下，宜…
- 时间紧迫：借助…外部环境，进一步…
- 措施须落实到具体机制/比例/范围，避免空洞

---

## 九、高辨识度语言习惯（对照自查用）

| 习惯 | 示例 |
|------|------|
| "叠加"并列多因素 | 叠加 11 月以来美联储降息预期升温、中美关系阶段性缓和等因素 |
| "凸显"揭示深层问题 | 凸显出美债市场机制的脆弱性 |
| "稳中有升/降"轻微变化 | 美元价值贮藏功能稳中有升 |
| "积极稳慎"政策建言固定修饰 | 积极稳慎推进人民币由"以汇为锚"向"以债为锚"的转变 |
| "若…可能…"风险情景 | 若高利率持续，可能引发较大亏损 |
| "短期看/长期看"双维判断 | 短期看，套利机会持续；长期看，应密切关注流出风险 |
| "以…为分水岭"历史分期 | 可粗略以 2008 年全球金融危机为分水岭，分两个阶段来看 |
| "尚未根本动摇"稳定性判断 | 其作为全球头号"安全资产"的地位尚未根本动摇 |
| "进一步"极高频 | 几乎每段政策建议均出现，如进一步提振增长前景 |
| "较高/较低水平"相对量化 | 仍处于较低水平（不用"很高"、"很低"） |
| 引号标注专有概念 | "量宽型"亏损、"冲销型"、"以汇为锚" |
| "不一而同" | 亏损机制和经济含义不一而同 |

## 十、自查清单（生成报告后必须逐项核对）

- [ ] 开篇摘要是否覆盖：现象 → 原因 → 影响 → 建议？
- [ ] 每个一级节标题是否能独立表达该节核心论点？
- [ ] 有无泛化标题（"背景"、"分析"、"总结"）？
- [ ] 政策建议节是否使用"一是…；二是…；三是…"分条表达？
- [ ] 措施是否落实到具体机制，避免空洞？
- [ ] 首次出现的机构缩写是否已注明全称？
- [ ] 数据引用是否精确到一位小数，估算是否加"约"？
- [ ] 有无"令人震惊"、"首先/其次/再次/最后"等禁用表达？
- [ ] 高辨识度词汇（叠加、凸显、稳中有升、积极稳慎、进一步）是否自然出现？
